# Path Gen Tick
[Header](https://tritonuas.github.io/wiki/software/obc/tick_architecture/ticks/pathgen/<https:/github.com/tritonuas/obcpp/blob/main/include/ticks/path_gen.hpp>) | [Source code](https://tritonuas.github.io/wiki/software/obc/tick_architecture/ticks/pathgen/<https:/github.com/tritonuas/obcpp/blob/main/src/ticks/path_gen.cpp>)
This tick handles the generation of the path. It essentially spawns up a background thread which does the path generation. It then routinely checks on the status of this thread, and once the path is generated transitions to `PathValidate`.
